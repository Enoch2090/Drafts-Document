# ActionGroup
代表一个动作组。这个对象可以用来搜索动作组，并在之后使用 `App` 这个对象中的方法来加载找到的动作组。
## 可调用的实例属性
- `name` _[动作组名字，只读]_
	- 返回动作组的名字。
## 可调用的类函数
- `getAll()` -\> _以列表形式返回动作组对象_
	- 获取所有可用的动作组。
- `find(name)` -\> _动作组_
	- 搜索指定名字的动作。如果找到则返回动作，如果未找到则返回「未定义」。

## 示例
```js
// 搜索动作组
var group = ActionGroup.find("Basic"); //搜索「Basic」这个动作组，并赋值给变量「group」
app.loadActionGroup(group);
```
